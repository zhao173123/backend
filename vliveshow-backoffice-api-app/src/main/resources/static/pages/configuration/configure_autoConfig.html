推荐频道自动置顶方案配置

  • 1、更新频率： 分钟 （App自动刷新一次，手动刷新可以获取最新的内容）
  • 2、显示规则： 现场规则 * % +现场规则 * 属性规则 * % = 排序积分
    • A 现场规则：= 观众数 * % + 打赏人数 * % (不是人次)
    • B 属性规则：= 高质量粉排名第一的主播记为
    • 排名第二的主播记为
    • 排名第三的主播记为
    • 排序积分最高的排在第一位，其他以此类推
  • 3、允许后台点选某个或者几个直播间主动排序，并且固定位置，剩下的主播遵从第2项规则排序
  • 4、新开直播间规则：注 在手动固定位置的N个直播间后，
    • 加 个正常置顶的直播间，之后再加 个新开直播间
    • 在开播 分钟内，如果观众数没有达到 人（app显示人数），则直接在{{step1Time==null ? 'X' : configInfo.step1Time}}分钟后自动退出置顶规则，
    • 如果观众数达到{{configInfo.step1Num==null ? 'X' : configInfo.step1Num}}人（app显示人数），则继续在置顶到 分钟，
    • {{configInfo.step2Time==null ? 'X' : configInfo.step2Time}}分钟后，如果观众数达到 人（含机器）则继续延长推荐{{configInfo.step2Time==null ? 'X' : configInfo.step2Time}}分钟。
    • 满 分钟后，不管直播数据如何，都退出置顶位置。
